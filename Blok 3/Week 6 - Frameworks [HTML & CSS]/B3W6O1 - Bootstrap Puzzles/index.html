Exercise

You're going to make 3 pages with the use of Bootstrap, by only adding classes to existing elements.
All 3 pages have the same HTML but a different layout.
The same technical requirements apply to each puzzle.
The second puzzle also has some functional requirements.

Technical Requirements

  • You may NOT edit the HTML code (except when adding classes)
  • You may NOT add your own CSS or style (through a CCS file or inline style)

Functional Requirements

  • The menu expands when you click on it.
  • For the menu icon you can make use of Font Awesome.
  • The images of the trees are shown as a carousel.
  • The card text is only visible after clicking "Read more …"
